# Description

Forest Rescue is a small 2D platformer prototype built using p5.js and p5play.  
The player controls a fox exploring a forest level to collect all leaves while avoiding enemies and hazards.

This Week 10 version focuses on polish and usability improvements, including:

- Menu screen
- Pause / Resume
- Save / Load system
- HUD elements
- Settings menu
- Restart and debug features

The goal of the game is to collect all leaves as fast as possible without losing all hearts.

Core mechanics:

- Move and jump platforming
- Collectible items
- Enemy hazards
- Timer tracking
- Health system

This sketch demonstrates how UI systems, menus, and state management can improve the player experience.

Bonus features:

- Settings menu (debug toggle, collision boxes, parallax, sound)
- Restart key and debug tools for testing
- Save system storing last run results

# Setup and Interaction Instructions

Run the game from GitHub Pages or open index.html in a browser.

Controls:

Arrow keys / WASD → Move  
Space / Up → Jump  
Z → Attack  
ESC → Pause / Menu  
R → Restart after death  
Enter → Start game from menu

Gameplay:

- Collect all leaves to win
- Avoid fire and enemies
- Losing all hearts = game over
- Timer records your run
- Save data keeps best result
